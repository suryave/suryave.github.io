About me

My name is Surya Venkatram and I'm a student at the Middlesex County Academy for Science, Mathematics, and Engineering Technologies. I like web design and development, programming in general, video games, and tennis. I attend hackathons and love learning new things.

A bit more about me



I'm a freshman at MCASMET, and was introduced to programming in my own home. I began to explore a bit of HTML/CSS, and then started to learn C++. Currently, I'm learning HTML5/CSS3, C++, C#, Javascript, and node.js. I have used many of these languages in hacks and projects that I've created for various hackathons.


Projects

Here are some projects that I've worked on:



SiteDit (Formerly FileDit) - A project created with my friend and classmate Kunal Adhia at CodeDay NJ Spring 2015. It uses a combination of node.js, HTML/CSS, and C++ to create a comprehensive solution that monitors the source code of a target site and when the source code is changed in any way alerts the user to the exact position where the changes were made and the changes made in the HTML code of that site.



Findr - A project created at HackRU Spring 2015 which attempted to take location data of nearby points of interest in varying categories based on the user's current location and display the data on the Pebble smartwatch. This was created in conjunction with my friends Ishan Arya, Sumant Pottepalem, and Amrit Chauhan.



Platform Game - A game created at CodeDay NJ Winter 2015 by my friends and classmates Ankit Patel, Poojit Hegde, Kunal Adhia, myself, Anthony Autera, and Mark Byrnes. This won the "best game award" at CDNJ.
